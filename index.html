About

This is an overview of my 4th year final project for the Interactive Digital Art & Design Course. The aim for the project was to create a visual identity and a website with appointment booking option for SETU Sports Injury Clinic on Carlow Campus.

Final project consists of:

Logo Design

Promotional materials such as poster, social media posts etc

Merchandise Mock-ups

Responsive website with light and dark mode

Appointment scheduling option on the website

Logo Design

Logo Sketch


Monochrome

Background Logo

Promotional Materials

Merchandise

Website

The website is a fully responsive Progressive Web App (PWA) that includes both light and dark mode options. It has an appointment booking function for the clinic and a contact form. Website was developed using HTML, CSS & JavaScript.
